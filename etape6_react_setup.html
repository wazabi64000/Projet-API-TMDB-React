1️⃣ Création du projet React

Exécutez :

npm create vite@latest frontend --template react
cd frontend
npm install
npm run dev

Vous avez maintenant un projet React fonctionnel, prêt à connecter votre API Node.js.

2️⃣ Structure du projet

  • src/App.jsx : composant principal
  • src/main.jsx : point d’entrée React
  • src/components/ : dossier pour vos composants réutilisables
  • index.html : page HTML racine

3️⃣ Explications pédagogiques

React permet de créer des composants modulaires et réutilisables. Chaque composant doit être simple et indépendant.

Exemple : MovieCard.jsx pour afficher un film. Métaphore : chaque carte = une fiche produit dans un magasin.

4️⃣ Exemple concret

function App() {
  return (
    <div>
      <h1>Hello React</h1>
    </div>
  );
}
export default App;

Tester ce composant pour vous assurer que le projet fonctionne correctement.

5️⃣ Prochaines étapes

Après cette étape, vous pourrez créer vos composants React, consommer l’API TMDB et construire la grille avec pagination.
