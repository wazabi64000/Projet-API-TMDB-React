Objectif

Créer des composants React réutilisables pour afficher les films.

Instructions

  1. Créer un dossier components/ dans src/.
  2. Créer le composant MovieCard.jsx pour afficher titre, image et date d’un film.
  3. Créer le composant MovieGrid.jsx pour organiser les films en grille.
  4. Créer un composant Pagination.jsx pour naviguer entre les pages.
  5. Importer ces composants dans App.jsx et tester leur rendu.

Explications pédagogiques

Chaque composant doit être simple et réutilisable. Métaphore : chaque carte = fiche produit, la grille = rayon du magasin, la pagination = ascenseur pour passer aux autres rayons.

Action concrète

src/components/MovieCard.jsx
export default function MovieCard({ movie }) {
  return (
    <div className="card">
      <img src={movie.poster} alt={movie.title} />
      <h3>{movie.title}</h3>
      <p>Sortie: {movie.release_date}</p>
    </div>
  );
}
