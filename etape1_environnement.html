Objectif

Installer Node.js, npm, et préparer le dossier de projet pour démarrer le développement.

Explications

Avant de coder, il faut un environnement stable. Node.js et npm permettent de gérer les modules nécessaires.

Exemple de code


# Créer le dossier projet
mkdir projet-tmdb
cd projet-tmdb

# Initialiser npm
npm init -y

Bonnes pratiques

  • Utiliser toujours la dernière version stable de Node.js
  • Initialiser le projet avec npm pour gérer les dépendances
  • Créer un fichier README.md dès le début

Notes et conseils

Vérifier que Node.js et npm sont installés correctement :

node -v
npm -v

Résumé

Vous avez créé votre dossier projet et initialisé npm. L'environnement est prêt pour installer les modules et démarrer le backend.
