Objectif

Installer les modules nécessaires pour le backend : express, axios, helmet, cors, dotenv, express-rate-limit.

Explications

Chaque module a un rôle précis :

  • express : créer le serveur HTTP
  • axios : faire des requêtes vers TMDB
  • helmet : sécuriser les headers HTTP
  • cors : gérer les accès cross-origin
  • dotenv : gérer les variables d'environnement
  • express-rate-limit : limiter le nombre de requêtes

Exemple de code

npm install express axios helmet cors dotenv express-rate-limit

Bonnes pratiques

  • Ne pas installer de modules inutiles
  • Ajouter les modules dans package.json
  • Utiliser un fichier .env pour les clés API

Notes et conseils

Vérifier que le package.json contient bien toutes les dépendances et la version correspondante.

Résumé

Les modules sont installés, l'environnement backend est prêt pour créer l’architecture MVC.
